## Launch Thread (10 tweets)

**1/10** I put a full Linux terminal inside Telegram. Then I connected it to Claude Code. Now I can build, deploy, and fix servers -- from my phone. Here's teletty. Open source. Free.

**2/10** The problem: Every developer has been here -- Server goes down at dinner. CI/CD fails on the train. Quick SSH fix needed, laptop is home. You pull out your phone and... nothing works.

**3/10** teletty fixes this. It's a Telegram Mini App that gives you: A real terminal session on your phone, smart buttons (no tiny keyboard typing), Claude Code integration built-in, one-tap approval for AI actions. All inside Telegram.

**4/10** Here's what it looks like in action: Open Telegram > Launch teletty > Ask Claude Code to fix the bug > Review the diff > Tap "Approve" > Done. From your phone. [attach video]

**5/10** Why Telegram? 1B monthly users already have it. Mini Apps = zero install. Push notifications built-in. Works on any phone, any OS.

**6/10** The Claude Code integration is the killer feature. You're running an AI coding agent that can read your codebase, write fixes, run tests, deploy changes. And YOU approve every action with one tap.

**7/10** Typing long commands on a phone = pain. teletty has smart context-aware buttons: Y/n confirmations, numbered options, Allow/Deny for AI actions. It's like a Stream Deck for your terminal.

**8/10** Real use cases: Emergency server fixes from anywhere. Reviewing PRs on the go. Running deployments from the couch. Monitoring logs while walking the dog.

**9/10** teletty is 100% open source. MIT licensed. Self-hostable. No telemetry. No vendor lock-in. Every PR is welcome. github.com/teletty/teletty
