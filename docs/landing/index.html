Features

Everything included

A purpose-built terminal for mobile. No compromises.

tmux

Tmux Sessions

Sessions survive disconnects. Close Telegram, come back later -- you're exactly where you left off.

mic

Voice Commands

Push-to-talk voice input. Dictate terminal commands when typing isn't practical.

tabs

Multi-Tab

Up to 4 concurrent sessions. Switch between servers or tasks instantly.

api

Emergency API

Management API via HTTPS. Execute commands when SSH is completely down.

auth

Telegram Auth

HMAC-SHA256 verification. No passwords, no keys -- Telegram handles it.

log

Audit Logging

Every session logged via tmux pipe-pane. Full accountability.

How it works

From Telegram to a running terminal in seconds.

1

Add the bot

Open @YourBot in Telegram and tap the menu button.

2

Authenticate

Telegram verifies you automatically. If whitelisted, you're in.

3

Full terminal

tmux session, smart buttons, voice -- everything works.

Up and running in 60 seconds

Choose your preferred method.

Give this to your AI agent (Claude Code, Cursor, etc.):

prompt 
Clone https://github.com/teletty/teletty, create .env from
.env.example, set BOT_TOKEN and ALLOWED_USER_IDS, then run
docker compose up -d. Verify the bot responds in Telegram.
bash 
git clone https://github.com/teletty/teletty && cd teletty
cp .env.example .env
# Edit .env: set BOT_TOKEN, ALLOWED_USER_IDS, SESSION_SECRET
docker compose up -d
bash 
git clone https://github.com/teletty/teletty && cd teletty
cp .env.example .env && nano .env
npm install && node server.js
